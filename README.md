# 北京語学習アプリ：効率化と自信・技術の定着

> **ここにあるのは、私がAndroid Studioでスマホアプリ（APK）を完成させるための『心臓部』となったWebアプリのコードです。制作の舞台裏は以下の通りです。**

### 【開発の経緯】

スワヒリ語版での苦闘を経て、私は一つの「型」を手に入れた。Geminiと交わした膨大な対話レスを紐解き、「Android Studioの使い方」「コマンドプロンプトの呪文」「index.jsの構造」を自らマニュアル化した。

「自分でもできるはずだ」という確信のもと、北京語版の作成に挑む。スワヒリ語での経験が血肉となり、あんなに苦労した工程が嘘のように、北京語アプリは驚くほどスムーズに、すんなりと形を成した。

複数のAIを使い分け、使い分けられながら、Pythonでのデータ抽出も成功。知識が「点」から「線」に繋がり、開発の楽しさが最高潮に達した瞬間を記録した作品。
